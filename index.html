Array prototype

El prototype un constructor global disponible para todos los objetos de JavaScript. prototype hace referencia al objeto global Array (). El prototype-constructor le permite agregar nuevas propiedades y métodos a las matrices. Al construir una nueva propiedad, TODAS las matrices obtendrán esta propiedad y su valor. Al construir un nuevo método, TODAS las matrices obtendrán este método.

Métodos

Un método es un bloque de código que contiene una serie de instrucciones. Un programa hace que se ejecuten las instrucciones al llamar al método y especificando los argumentos de método necesarios.

toString

El método JavaScript toString() convierte una matriz en una cadena de valores de matriz (separados por comas)

const fruits = ["Banana", "Orange", "Apple", "Mango"]; document.getElementById("demo").innerHTML = fruits.toString();

join

El método join() también une todos los elementos de la matriz en una cadena. Se comporta igual toString(), pero además puedes especificar el separador:

const fruits = ["Banana", "Orange", "Apple", "Mango"]; document.getElementById("demo").innerHTML = fruits.join(" * ");

pop

El método pop() elimina el último elemento de una matriz:

const fruits = ["Banana", "Orange", "Apple", "Mango"]; fruits.pop(); // Removes "Mango" from fruits

El método pop() devuelve el valor que "apareció":

const fruits = ["Banana", "Orange", "Apple", "Mango"]; let x = fruits.pop(); // x = "Mango"

push

El método push() agrega un nuevo elemento a una matriz (al final):

const fruits = ["Banana", "Orange", "Apple", "Mango"]; fruits.push("Kiwi"); // Adds "Kiwi" to fruits

El método push() devuelve la nueva longitud de la matriz:

const fruits = ["Banana", "Orange", "Apple", "Mango"]; let x = fruits.push("Kiwi"); // x = 5

shift

El método shift() elimina el primer elemento de la matriz y "desplaza" todos los demás elementos a un índice más bajo.

const fruits = ["Banana", "Orange", "Apple", "Mango"]; fruits.shift(); // Removes "Banana" from fruits

El método shift() devuelve el valor que se "desplazó":

const fruits = ["Banana", "Orange", "Apple", "Mango"]; let x = fruits.shift(); // x = "Banana"

unShift

El método unshift() agrega un nuevo elemento a una matriz (al principio) y "deshace" los elementos más antiguos:

const fruits = ["Banana", "Orange", "Apple", "Mango"]; fruits.unshift("Lemon"); // Adds "Lemon" to fruits

El método unshift() devuelve la nueva longitud de la matriz.

const fruits = ["Banana", "Orange", "Apple", "Mango"]; fruits.unshift("Lemon"); // Returns 5

splice

El método splice() se puede utilizar para agregar nuevos elementos a una matriz:

const fruits = ["Banana", "Orange", "Apple", "Mango"]; fruits.splice(2, 0, "Lemon", "Kiwi");

El primer parámetro (2) define la posición en la que se deben agregar (empalmar) nuevos elementos . El segundo parámetro (0) define cuántos elementos deben eliminarse . El resto de parámetros ("Limón", "Kiwi") definen los nuevos elementos a añadir . El splice()método devuelve una matriz con los elementos eliminados:

const fruits = ["Banana", "Orange", "Apple", "Mango"]; fruits.splice(2, 2, "Lemon", "Kiwi");
